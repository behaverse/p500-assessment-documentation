Parameters

Input data (2 parameters)

PeripheralTask

If the peripheral task consists of an identification task or a detection task or nothing.

Type: enum Default: Detection
Note: central & peripheral tasks cannot both be None

CentralTask

If the central task consists of an identification task or nothing.

Type: enum Default: Identification
Note: central & peripheral tasks cannot both be None

Stimulus parameters (10 parameters)

FixationPointDisplayDuration

How long we display a fixation point before any stimulus. If equal to 0, will not display anything. Timing starts from the beginning of the trial. This duration also affects when the first stimulus is shown (i.e., acts like a foreperiod).

Type: float Default: 1 Range: 0 - Inf

CentralStimulusSymbolIndex

There are two possible stimuli that can be shown on any given trial; this variable says which one of the two it was.

Type: int Default: Uniform random sampling Range: 1 - 2

CentralStimulusOnset

Determines when the central stimulus is shown. The duration starts after the end of the fixation period.

Type: float Default: 0.1 Range: 0 - Inf

CentralStimulusDisplayDuration

How long the central stimulus is shown.

Type: float Default: 0.2 Range: 0 - Inf

PeripheralStimulusSymbolIndex

When PeripheralTask == DetectionAndIdentification, there are two possible stimuli that can be shown on any given trial (typically different from the stimuli shown for the central task); this variable says which one of the two it was. (1 is out, 2 is in).

Type: int Default: Uniform random sampling Range: 1 - 2
Note: When PeripheralTask == Detection, this vaiable is always set to 1.

PeripheralStimulusPositionIndex

Indicate where the peripheral stimulus was shown, out of the 8 possible positions (see General Layout). Although there are 3 circles on which stimuli could appear (and distractor do appear) we only use the outermost radius to display stimuli and hence there are only 8 positions.

Type: int Default: Uniform random sampling Range: 1 - 8
Note: The disk on which stimuli are presented in subdivided in 8 pizza-slices; the upper center-right is 1, the next on its right is 2, the next on it's right is 3, etc. until reachint the top center-left value of 8 which is to the left of 1.

PeripheralStimulusOnset

Determines when the peripheral stimulus is shown. The duration starts after the end of the fixation period.

Type: float Default: 0.1 Range: 0 - Inf

PeripheralStimulusDisplayDuration

How long the peripheral stimulus is shown.

Type: float Default: 0.2 Range: 0 - Inf

ShowPeripheralDistractors

If there are distractors in the periphery or not. If True, all non-stimulus locations are filled by distractors. If False, none are.

Type: boolean Default: "false"

MaskMinDisplayDuration

Minimum time during which the masks are displayed. Masks are displayed independently after each stimulus has finished being displayed. All masks are removed at the same time, after each mask has been displayed for at least MaskMinDisplayDuration (see “Timing” above).

Type: float Default: 0.5 Range: 0 - Inf

Feedback parameters (2 parameters)

ShowAccuracyOnButtonFeedback

Whether the button highlight shows if the user response is correct/incorrect or instead does not reveal response accuracy. If set to True, the highlight is green for correct responses and red for incorrect responses.

Type: boolean Default: "false"
Note: We want to show the feedback per button only after all the responses were given, NOT immediately after each response.

ShowAccuracyCheckFeedback

Whether we should show the valid or wrong “check” feedback at the end of a trial; this also appears always after the button feedback when ShowAccuracyOnButtonFeedback == True.

Type: enum Default: Always
