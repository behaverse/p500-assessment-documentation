Parameters

Cue parameters (4 parameters)

EndogenousCueType

The type of endogenous cue that will be displayed to the user on the screen center.

Type: enum Default: Arrow

EndogenousCueValidity

Qualifies the type of information given by the cue.

Type: enum Default: Valid
Note: “Valid” will be used for the pro-saccade condition and “Invalid” for the anti-saccade condition. “NoGo” cue makes sense only in some specific context. If a user doesn’t press on a NoGo trial it will currently not be considered correct.

ExogenousCueType

The type of exogenous cue that will be displayed at potential stimulus locations.

Type: enum Default: Circle

ExogenousCueValidity

Qualifies the type of information given by the cue.

Type: enum Default: Valid

Stimulus parameters (5 parameters)

StimulusType

The type of stimulus that will be displayed (i.e., this is not the cue).

Type: enum Default: Number
Note: Button is only available in classic SAC, all others only in RSAC.

MatrixSize

The size of the matrix (N), if StimulusType = Matrix. If N = 3, the matrix will be 3x3 and will have 9 cells.

Type: integer Default: 3 Range: 2 - 5

StimulusValue

The value of the stimulus.

Type: integer Default: 1 Range: 0 - Inf
Note: If StimulusType == Number, it’s the number that will be shown. If StimulusType == Symbol, it’s the index in the symbol list (indice starts from 0 and ends at 9). If StimulusType == VerticalArrow | HorizontalArrow, it’s the index in the symbol list (indices start from 1 end at 2). If StimulusType == Matrix, the number of lit cells. If there are more lit cells than the total number of cells, all cells are lit. If StimulusType == Button, this parameter is ignored.

StimulusPosition

The position of the stimulus.

Type: enum Default: Center

ShowDistractor

Whether we want to show a distractor where the stimulus is not (for example “↑“ on the right if the stimulus is on the left).

Type: boolean Default: "false"
Note: If the stimulus is in the center, there can’t be any distractor.

Timing parameters (13 parameters)

FixationPointOnset

When the fixation point should be shown in seconds relative to trial onset.

Type: float Range: 0 - Inf

FixationPointOffset

When the fixation point should be hidden in seconds relative to the first endogenous or exogenous cue onset (or stimulus or user input if both cues are hidden).

Type: float Range: -Inf - inf

TaskCueOnset

When the task cue should be shown in seconds relative to trial onset.

Type: float Range: 0 - Inf

TaskCueDisplayDuration

The time the task cue stays on screen.

Type: float Range: 0 - Inf

EndogenousCueOnset

When the endogenous cue should be shown in seconds relative to the stimulus onset.

Type: float Default: -0.5 Range: -Inf - inf
Note: Because the endogenous cue appears before the stimulus, this value is negative.

EndogenousDisplayDuration

The time the endogenous cue stays on screen.

Type: float Default: 0.4 Range: 0 - Inf

ExogenousCueOnset

When the exogenous cue should be shown in seconds relative to the stimulus onset.

Type: float Default: -0.5 Range: -Inf - inf
Note: Because the exogenous cue appears before the stimulus, this value is negative.

ExogenousCueDisplayDuration

The time the exogenous cue stays on screen.

Type: float Default: 0.4 Range: 0 - Inf

StimulusOnset

When the stimulus should be shown in seconds relative to trial onset.

Type: float Default: 0.5 Range: 0 - Inf

StimulusDisplayDuration

The time the stimulus stays on screen.

Type: float Default: 0.15 Range: 0 - Inf

NoiseMaskOnset

When the noise mask should be shown in seconds relative to the end of the stimulus shown (onset + duration). In other words, the delay after the stimulus has been hidden.

Type: float Default: 0.05 Range: 0 - Inf

NoiseMaskDisplayDuration

The display duration of the noise mask displayed on top of the stimulus.

Type: float Default: 0.15 Range: 0 - Inf

UserInputOnset

When the user can input his answer in seconds relative to the stimulus onset.

Type: float Default: 0.5 (1 in RSAC) Range: 0 - Inf

User Input parameters (1 parameters)

MaxResponseTime

The maximum time the user can take to answer. After that, the trial is considered failed.

Type: float Default: 5 Range: 0.01 - Inf

Feedback parameters (2 parameters)

ShowAccuracyCheckFeedback

Whether we should show the valid or wrong “check” feedback at the end of a trial.

Type: enum Default: Always

ShowCorrectTrialResponseFeedback

Whether we should show the correct response feedback at the end of a trial.

Type: enum Default: Never
