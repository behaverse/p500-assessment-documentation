Glossary

Box an interactive element, with a hidden state. The user can click on any box to reveal its state.
Token a symbol hidden in one of the boxes.
Trial corresponds to a configuration of boxes and an order of tokens. A trial ends if user finds all tokens or fails.
SearchEpisode a trial is composed on several searchEpisodes (at most as many as there are tokens to be found). Each time the user finds a new token a new searchEpisode is triggered and it is now OK to reopen a box that was previously clicked on and did not contain a token.
State the state of the box, an enum in { NewEmpty, OldEmpty, NewToken, OldToken }):
NewEmpty The box is empty.
OldEmpty The box is empty and has already been clicked during this searchEpisode.
NewToken The box contains the current token to be found.
OldToken The box contains a token the user already found in this trial (during a previous searchEpisode)

Parameters

Constants (1 parameters)

MaxBoxCount

The maximum number of boxes that can be presented to the user in a given trial. This is determined empirically to avoid cluttering the screen.

Type: integer Default: 25

Stimulus parameters (3 parameters)

GridSize

The size of the matrix.

Type: integer Default: 5 Range: 3 - 5

BoxCount

The number of boxes presented to the user.

Type: integer Default: 4 Range: 1 - MaxBoxCount

TrialEndCondition

Defined when the trial ends depending on user actions.

Type: enum Default: OnError

User input parameters (2 parameters)

AvoidFirstTryTokens

Whether we want to avoid lucky guesses when the player opens the good box on its first click in a search episode. If so, the token is moved in another unexplored box instead of being found in the currently searched box.

Type: enum Default: Never

MaximumTrialDuration

The maximum allowed time to complete the trial (find all tokens). After that, the trial is considered failed.

Type: float Default: 999 Range: 1 - Inf

Feedback parameters (3 parameters)

ShowAccuracyCheckFeedback

Whether we should show the accuracy check feedback at the end of a trial.

Type: enum Default: Never

ShowPastTokensCondition

Shows the tokens that have been found during this trial.

Type: enum Default: OnError

RevealBoxesOnError

Reveal the content of all boxes when the user makes an error.

Type: boolean Default: "false"
